
## 📋 Описание проекта

**Железнодорожная система управления** - это программная система на языке Python, моделирующая работу железнодорожного транспорта. Система предоставляет интерфейс командной строки (CLI) для управления движением поездов, обслуживания станций, технического обслуживания подвижного состава, продажи билетов и контроля безопасности.

### 🎯 Предметная область

Проект разработан в рамках курса "Проектирование программного обеспечения интеллектуальных систем" и моделирует организацию и управление железнодорожным транспортом.

### 📦 Важные сущности

- **Железнодорожные пути** - инфраструктура для движения поездов
- **Станции** - остановочные пункты с платформами
- **Поезда** - подвижные составы из локомотива и вагонов
- **Локомотивы** - тяговые единицы
- **Вагоны** - пассажирские или грузовые единицы
- **Расписание** - временные интервалы прибытия/отправления
- **Билеты** - документы на проезд

### ⚙️ Основные операции

1. **Операция движения поездов** - перемещение составов по путям
2. **Операция обслуживания станций** - управление станционной инфраструктурой
3. **Операция технического обслуживания** - ремонт и диагностика подвижного состава
4. **Операция продажи билетов** - покупка и отмена билетов
5. **Операция контроля безопасности** - проверка перед отправлением
